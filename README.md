# YouTube Clone (Frontend UI)

A responsive, pixel-perfect frontend replica of the YouTube homepage. This project focuses on modern web design techniques, including CSS Grid, Flexbox, and responsive layout handling for different screen sizes.

## 🚀 Features

- **Responsive Design**: Adapts seamlessly between desktop, tablet, and mobile devices using CSS media queries.
- **Custom Header**: Includes a search bar, create button, and branding logo.
- **Navigation Sidebar**: Dedicated sidebar with Home, Subscriptions, and Explore categories.
- **Video Grid**: A dynamic grid system that rearranges video thumbnails based on the viewport width.
- **Hover Effects**: Interactive states on video thumbnails and navigation items for better user engagement.

## 🛠 Tech Stack

- **HTML5**: Structured semantic markup for the web interface.
- **CSS3**: Layout, styling, and animations.
- **Google Fonts**: Utilizes "Roboto" for a clean, modern aesthetic.
